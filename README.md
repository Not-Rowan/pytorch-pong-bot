# pytorch-pong-bot
pong bot that i made using pytorch. it's basically a copy of my C one

To run the game:

1. first run pongBot.py (all settings are preset to work out of the box)
2. follow the instructions from the program
3. run game.py in another terminal window or tab
4. wait for the bot to train or watch it playing (if you can't see the game, see below)

IMPORTANT: I have a file called render.txt which has the number 1 or 0 in it. this controls whether the game is rendered to the screen. If you cannot see the game and you would like to, put the number 1 in this file, otherwise, put a number 0. (if you don't render the game while training, it'll run much faster :) )
